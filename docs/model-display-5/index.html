Comment retrouver une couche particulière ?

On peut accéder à 1ère couche avec :

model.layers[0]

Je t'en ai parlé déjà, on peut donner un nom à une couche, ce qui permet d'avoir un nom fixe.
Dans ce cas là, on peut récupérer la couche de la manière suivante :

model.get_layer("input")

Et tu peux retrouver le format d'entrée avec l'attribut input_shape.

model.get_layer("input").input_shape

Cela donne :

(None, 784)

On retrouve bien le format d'entrée 784. Le None ici correspond à la taille du Batch Size, qui n'est pas connu à l'avance.
